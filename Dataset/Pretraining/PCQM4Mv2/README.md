PCQM4Mv2 is an important dataset in the Open Graph Benchmark (OGB). Developed with contributions from the DeepMind team and the broader research community, it focuses on quantum chemical property prediction tasks and provides standardized evaluation metrics for benchmarking molecular machine learning models.  
The PCQM4Mv2 dataset is available on https://ogb.stanford.edu/docs/lsc/pcqm4mv2/.
