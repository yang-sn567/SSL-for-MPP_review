MoleculeACE, which is a benchmark testing tool specifically designed to evaluate the performance of machine learning models in the active cliff prediction task. Activity cliffs refer to pairs of molecules that are highly similar in chemical structure but exhibit large differences in biological or pharmacological activity. Typically, one molecule shows strong activity while the other has very weak or negligible activity. These activity cliff pairs present a major challenge for molecular property prediction models, as accurately predicting them requires a deep understanding of subtle structure–activity relationships.  
The MoleculeACE dataset is available on https://github.com/molML/MoleculeACE.
